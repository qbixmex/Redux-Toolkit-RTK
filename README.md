# Redux Toolkit tutorial

This is a Redux Toolkit tutorial for React 18

## Install Dependencies

### YARN

```
yarn
```

### NPM

```
npm insall
```

## Development Mode

### YARN

```
yarn dev
```

### NPM

```
npm run dev
```

## Make a build

### YARN

```
yarn build
```

### NPM

```
npm run build
```

## Testing

### YARN

```
yarn test --watch
```

### NPM

```
npm run test --watch
```
